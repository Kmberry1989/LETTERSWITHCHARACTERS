# **App Name**: Letters with Characters

## Core Features:

- Multiplayer Scrabble: Enable turn-based Scrabble games with multiple players.
- Customizable Avatars: Allow users to create and customize their animal character avatars with unlockable swag like hats, props, tops, and other visuals.
- Themed Boards & Tiles: Offer a variety of visual themes and tile designs, some earned through goals.
- Leaderboard: Track and display player rankings.
- Sticker Reactions: Enable players to send stickers during matches.
- Word Search Minigame: Include a word search minigame for additional engagement.
- AI Assistant for Word Suggestions: Provide an AI-powered tool that suggests valid words based on the current board state and the player's tiles.
- Login with Google: Allow users to log in with their Google account

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a classic board game feel.
- Background color: Light gray (#EEEEEE) to ensure readability and a clean interface.
- Accent color: Bright yellow (#FFEB3B) to highlight interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use simple, clean icons for navigation and features.
- Organize the game board clearly with distinct sections for tiles, scores, and chat.
- Use subtle animations for tile placement and scoring to provide feedback.
